How to Extract Data

Create an Excel file with the following columns:

  • field_name - A descriptive name for the data you want to extract
  • search_pattern - The text pattern to search for in the document
  • extraction_type (optional):
    • exact - Find the exact search pattern
    • regex - Use regular expression pattern matching
    • after_pattern - Extract text that follows the pattern
    • nlp - Use natural language instructions to extract information
  • context_before (optional) - Text that should appear before the pattern
  • context_after (optional) - Text that should appear after the pattern
  • instructions (optional) - Natural language instructions or additional context (important for NLP extraction)

Choose which documents you want to extract data from:

  • Use the checkboxes to select individual documents
  • Use "Select All" to choose all available documents
  • You can select multiple documents for batch extraction

To use the NLP extraction feature:

  • Set extraction_type to nlp in your Excel file
  • In the instructions column, provide a clear description of what to extract, such as:
    • "Extract the invoice total amount"
    • "Find the customer's full address"
    • "Extract the shipping date from this document"
    • "Locate the policy number in this insurance document"
  • The search_pattern field can be used to provide additional context if needed
  • The system will analyze the document semantically to find the information you requested
  • Unlike pattern matching, NLP extraction can understand the meaning of content and identify relevant information even when the exact wording varies

After extraction, you'll see a table with your results:

  • Each row represents a found match in your documents
  • Results include the document name, page number, and extracted value
  • Context around each match is provided to verify accuracy
  • You can export the results to CSV for further analysis
